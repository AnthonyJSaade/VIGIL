# Vibe Notes API

A lightweight personal notes service whipped up with AI assistance over a weekend.

## Features

- Register / login with a password (hashed for safety!)
- Save and search notes
- Import notes from a backup file
- Fetch notes directly from any URL
- View notes with nice HTML formatting

## Getting Started

```bash
pip install -r requirements.txt
python app.py
```

The API will be running on `http://localhost:5001`.

## Tech Stack

- Flask
- SQLite
- `requests` for remote note fetching
