# Vibe File Share

A simple service for uploading files and sharing them with a short link. Built quickly with AI assistance.

## Features

- Upload any file and get a shareable slug
- Download files with a short URL
- Auto-generate thumbnails via ImageMagick
- Mirror any remote URL through the server
- Admin delete endpoint protected with an API key

## Getting Started

```bash
npm install
npm start
```

The API runs on `http://localhost:3002`.

## Tech Stack

- Express.js
- Multer for multipart uploads
- ImageMagick's `convert` for thumbnails
